================================================================
         일류 손익 대시보드 — 사용 가이드
================================================================

■ 접속 주소
https://illryu-dashboard.vercel.app

■ 접속 방법
이메일 + 비밀번호 로그인 (계정은 관리자가 발급)


================================================================
1. 데이터 구조
================================================================

[구글 시트] → 당월 실시간 데이터 입력 공간
     ↓ 마감 버튼 클릭 시
[Supabase DB] → 과거 마감 데이터 영구 보관
     ↓
[대시보드] → 화면에 표시

  ┌────────────────┬────────────────┬────────────────────┐
  │ 구분           │ 데이터 위치    │ 조회 방법          │
  ├────────────────┼────────────────┼────────────────────┤
  │ 당월 데이터    │ 구글 시트      │ 실시간 버튼 클릭   │
  │ 과거 데이터    │ DB             │ 년/월 필터 선택    │
  └────────────────┴────────────────┴────────────────────┘


================================================================
2. 월별 업무 흐름
================================================================

① 매월 초  : 구글 시트에 당월 데이터 입력 시작

② 매월 중  : [📡 구글 시트 실시간] 버튼으로 수시 확인

③ 매월 말  : [🗓️ 이번 달 마감] 버튼 클릭
              → 시트 데이터가 DB에 영구 저장됨
              → 다음 달 시트 새로 작성 시작

④ 다음 달  : 이전 월은 DB에서 년/월 필터로 언제든 조회 가능


================================================================
3. 상단 버튼 기능
================================================================

[📡 구글 시트 실시간]
  → 현재 구글 시트 데이터를 즉시 불러와 화면에 표시
  → 빨간 "LIVE" 표시 나타남

[📊 DB 보기로 전환]
  → 실시간 모드 종료, DB 데이터로 복귀

[💾 보고서 저장]
  → 현재 필터 조건과 집계 요약을 DB에 기록

[🗓️ 이번 달 마감]
  → 구글 시트 데이터를 DB에 영구 저장 (월 마감 처리)
  → 마감 후에는 DB에서 조회 가능

[🔄 새로고침]
  → 데이터 다시 불러오기

[📄 PDF 다운로드]
  → 현재 화면을 PDF 파일로 저장


================================================================
4. 필터 사용법
================================================================

[년도]  전체 / 2024 / 2025 / 2026
[월]    전체 / 1월 ~ 12월
[주차]  전체 / 1주차 ~ 5주차 (한국 기준 월요일 시작)
[일]    전체 / 1일 ~ 31일
[구분]  다중 선택 가능
        - 접수형 / 관리형 / 보장형 / 테스트 / 보장완료 / 중단건
        - "전체 선택" 버튼: 모두 체크
        - "전체 해제" 버튼: 모두 해제
[검색]  업체명 / 상호명 / 담당자 부분 검색

※ 로그인하면 현재 한국 날짜 기준 당월이 자동 설정됩니다.
※ 필터를 변경하면 KPI 카드, 차트, 리스트, PDF 모두
   동일 기준으로 자동 연동됩니다.

5. 화면 구성
================================================================

[KPI 카드 4개]
  - 매출 합계 / 매입 합계 / 순익 합계 / 손익률
  - 전월 대비 증감(%) 표시

[주차별 순익 차트]
  - 주차별 막대 차트
  - 보라색 = 흑자 / 빨간색 = 적자

[업체별 순익 차트]
  - 상위 15개 가로 막대 차트
  - [전체 보기] 버튼으로 전체 업체 확인 가능

[일자별 손익 추이]
  - 매출/매입/순익 꺾은선 차트

[구분별 손익 요약]
  - 접수형, 관리형 등 구분별 집계 테이블

[적자 업체 TOP 5]
  - 손실이 큰 업체 순위

[업체별 상호명 분포]
  - 대행사별 상호명 순익 분포 피벗 테이블

[상세 거래 리스트]
  - 전체 거래 내역
  - 정렬 / 검색 / 수정 / 추가 가능


================================================================
6. 거래 수정 / 추가 방법
================================================================

■ 기존 거래 수정
  1. 상세 거래 리스트에서 수정할 행 클릭
  2. 팝업에서 항목 수정 (날짜/담당자/업체명/구분/매출/매입)
  3. 순익은 자동 계산 (매출 - 매입)
  4. [저장] 클릭 → 즉시 반영

■ 누락 거래 추가
  1. 상세 거래 리스트 우측 상단 [+ 거래 추가] 버튼 클릭
  2. 정보 입력 후 [저장]

■ 구분 일괄 변경
  1. 변경할 행 체크박스 선택 (복수 선택 가능)
  2. 나타나는 입력란에 새 구분명 입력
  3. [✓ 변경 적용] 클릭

================================================================
8. 계정 관리 (관리자)
================================================================

■ 새 계정 추가
  1. supabase.com 접속 → 프로젝트 선택
  2. 좌측 Authentication → Users
  3. Add User → 이메일/비밀번호 입력
  4. Email Confirm 체크 후 저장

■ 코드 배포
  GitHub main 브랜치 push 시 Vercel 자동 배포
  수동: Vercel 대시보드 → Deployments → Redeploy


================================================================
9. 문제 해결
================================================================

  증상                      │ 해결 방법
  ─────────────────────────────────────────────────────────
  로그인이 안 됨             │ 관리자에게 계정 발급 요청
  실시간 데이터 안 불러와짐  │ 구글 시트 공유 설정 확인
  데이터가 비어 있음         │ 필터 초기화 버튼 클릭
  마감 후 데이터가 0건       │ 구글 시트 내용 확인 후 재시도
